# Allen C Fredrick - Professional Resume Website

A modern, responsive resume website showcasing my professional experience, skills, and projects.

## Features
- Modern, responsive design with animated hero section
- Detailed work experience timeline
- Skills and competencies showcase
- Project portfolio
- Contact form
- Particle.js background effects

## Pages
- Home (Professional Summary)
- Experience
- Education
- Skills
- Projects
- Contact

## Technologies Used
- HTML5
- CSS3 (Custom animations and transitions)
- JavaScript
- Particle.js
- Font Awesome
- Google Fonts

## Setup for Local Development
1. Clone this repository
2. Open index.html in your web browser
3. Navigate through different sections using the navigation menu

## GitHub Pages Setup
1. Go to repository Settings
2. Navigate to Pages section
3. Select main branch as source
4. Save the settings

Your site will be published at: https://[your-username].github.io/resume-website/
